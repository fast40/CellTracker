Step 1: Prepare Images

First, make sure that your images meet the following criteria:

Format

You should have one before and one after image in each experiment.

File Names

Each image in a before/after pair should start with a unique identifier

  • There should be exactly one before and one after image in each experiment.
  • Each image in a before/after pair should start with a unique identifier.
    • For example, each filename might begin with "experiment7" or "002" or similar.
    • Just make sure that both the before and after image start with the same unique identifier.
